== Description ==

Display your PageRank on your WordPress website. Forget about other complicated PageRank plugins that require the modification of templates with additional PHP code. This plugin simplifies the display of your PageRank by simply displaying it as a widget. Display your PageRank at a top-domain level or an individual page level.

== Installation ==

1. Upload `pagerank-widget.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Appearance > Widgets in WordPress
4. Select the section where you want the widget to appear and click Show
5. Click the Add button next to the PageRank Widget
6. Edit the option to display at a top-level or page level
7. Click save changes
